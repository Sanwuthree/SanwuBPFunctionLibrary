#SanwuBPFunctionLibrary

======================

##关于SanwuBPFunctionLib
*`SanwuBPFunctionLibrary`是一个UE4 蓝图函数库。

因为有些函数只能用C++实现。蓝图无法实现，所以在下写了这个插件。用于扩展BP的功能

目前只实现了`获取项目单例`对象。

以后会慢慢增加，近期目标是增加 `Http` 和 `Socket` 的支持


同时也希望大家能参与到这个项目里..  共同学习，共同进步。
